# 内容策略技能

## 适用场景
- 各品牌月度/周度内容规划与选题策划
- 热点话题捕捉与内容创意建议
- 竞品内容对标分析与差异化策略
- 爆款内容复盘与方法论沉淀

## 标准流程
1. **热点扫描**：收集当周/当月行业热点、社会热点、平台热门话题
2. **受众分析**：结合各品牌目标受众画像和内容消费偏好
3. **内容日历**：制定选题计划（选题+内容形式+发布节点+预期指标）
4. **Brief输出**：详细内容Brief（标题、脚本大纲、视觉方向、BGM建议）
5. **数据追踪**：发布后72小时数据跟踪，及时调整策略

## 品牌内容定位与调性

### 广汽传祺
- **调性**：家用实用 + 商务稳重，接地气不low
- **核心受众**：30-45岁家庭用户、中小企业主
- **内容方向**：
  - 家庭用车场景（接送孩子、周末出游）
  - 商务接待场景（MPV车型重点展示）
  - 性价比对比（同价位配置优势）
  - 真实车主证言与口碑传播
- **内容形式**：短视频60% + 图文30% + 直播10%
- **发布节奏**：每日2-3条，集中在12:00/18:00/20:00

### 上汽奥迪
- **调性**：豪华质感 + 生活方式，高级不冰冷
- **核心受众**：35-50岁高净值人群、企业高管
- **内容方向**：
  - 品质生活方式（高尔夫、品酒、艺术展）
  - 科技与工艺（灯厂技术、quattro四驱）
  - 车主圈层文化（奥迪车友会、私享活动）
  - 品牌历史与传承
- **内容形式**：短视频50% + 图文40% + 直播10%
- **发布节奏**：每日1-2条，集中在19:00/21:00

### 广汽昊铂
- **调性**：科技先锋 + 高端新能源，前卫不浮夸
- **核心受众**：28-40岁科技新贵、新能源早期采用者
- **内容方向**：
  - 智能驾驶技术深度解读
  - 超充/换电技术科普
  - 极致性能体验（赛道日、加速测试）
  - 科技美学（设计语言、内饰解读）
- **内容形式**：短视频55% + 图文25% + 直播20%
- **发布节奏**：每日2-3条，集中在12:00/18:00/21:00

### 广汽埃安
- **调性**：智能潮流 + 年轻活力，潮酷不低幼
- **核心受众**：25-35岁年轻家庭、科技爱好者
- **内容方向**：
  - 智能座舱玩法（车机交互、语音助手）
  - 潮改文化（改装案例、贴膜方案）
  - 生活场景（露营、自驾、城市通勤）
  - 技术普惠（弹匣电池、夸克电驱）
- **内容形式**：短视频60% + 图文20% + 直播20%
- **发布节奏**：每日2-3条，集中在12:00/17:00/20:00

## 爆款内容公式

### 短视频爆款要素
```
高完播率 = 前3秒钩子 + 节奏紧凑 + 音乐卡点 + 结尾引导
高互动率 = 争议性话题 + 提问式结尾 + 评论区运营
高转发率 = 实用干货 + 情感共鸣 + 社交货币
```

### 选题方法论
| 类型 | 占比 | 示例 |
|------|------|------|
| 热点借势 | 20% | 节假日、社会热点、平台挑战赛 |
| 产品种草 | 35% | 车型亮点、配置解读、对比测评 |
| 实用干货 | 25% | 购车攻略、用车技巧、养车知识 |
| 情感共鸣 | 15% | 车主故事、提车仪式、家庭场景 |
| 互动营销 | 5% | 有奖互动、直播抽奖、话题挑战 |

## 热点追踪机制
- **日常监控**：微博热搜、抖音热榜、小红书趋势（每日 09:00 / 14:00 / 19:00）
- **行业热点**：汽车之家/懂车帝/易车 首页推荐（每日 10:00）
- **竞品动态**：竞品账号新发内容自动抓取（HEARTBEAT 定时任务）
- **借势时效**：热点发生后 2 小时内完成选题判断，4 小时内出内容

## 输出格式
- **内容日历**：Excel/Markdown 表格，含完整排期
- **选题Brief**：单条内容的详细执行文档（标题、脚本、视觉、BGM）
- **爆款复盘**：对播放量 >10000 的内容进行拆解分析
