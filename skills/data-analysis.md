# 数据分析技能

## 适用场景
- 各品牌新媒体平台（抖音/快手/小红书/视频号）数据采集与分析
- 粉丝增长趋势、互动率、转化率等多维度数据分析
- 投放效果ROI评估与优化建议
- 跨品牌、跨平台横向对比分析

## 标准流程
1. **数据采集**：从各平台后台导出原始数据，或通过API拉取
2. **数据清洗**：去重、补缺、格式统一、异常值标记
3. **多维度分析**：按品牌/平台/时间/内容类型交叉分析
4. **可视化呈现**：折线图（趋势）、柱状图（对比）、热力图（时间分布）
5. **结论输出**：核心发现 + 数据支撑 + 优化建议

## 核心指标体系

### 流量指标
| 指标 | 计算方式 | 健康阈值 |
|------|---------|---------|
| 曝光量 | 平台直接提供 | 环比波动不超过 ±30% |
| 播放量 | 平台直接提供 | 完播率 >30% 为优质 |
| 主页访问量 | 平台直接提供 | 持续正增长 |

### 互动指标
| 指标 | 计算方式 | 健康阈值 |
|------|---------|---------|
| 互动率 | (点赞+评论+转发+收藏)/曝光量 | >5% 为优秀 |
| 点赞率 | 点赞数/曝光量 | >3% 为良好 |
| 评论率 | 评论数/曝光量 | >0.5% 为良好 |
| 转发率 | 转发数/曝光量 | >1% 为优秀 |
| 粉丝净增 | 新增关注-取关 | 日均正增长 |

### 转化指标
| 指标 | 计算方式 | 健康阈值 |
|------|---------|---------|
| CPL（线索成本） | 投放总费用/有效线索数 | 各品牌单独设KPI |
| 有效线索率 | 有效线索/总线索 | >60% |
| 到店转化率 | 到店数/有效线索数 | >15% |
| 成交转化率 | 成交数/到店数 | >25% |

## 分析方法库

### 趋势分析
- 7日/30日/90日滚动均值对比
- 同比（与去年同期）和环比（与上周期）双维度
- 异常波动自动标记（波动幅度 >2σ 触发告警）

### 内容分析
- 按内容形式（短视频/图文/直播）拆分表现
- 爆款因子分析：发布时间、话题标签、BGM、时长
- 低效内容识别与淘汰建议

### 竞品对标
- 同城同品牌经销商排名
- 竞品品牌账号横向对比
- 行业大盘基准线参考

## 异常判定规则
| 异常类型 | 判定条件 | 处理动作 |
|---------|---------|---------|
| 流量突降 | 单日曝光环比下降 >50% | 检查是否限流/违规 |
| 互动异常 | 互动率 >20% 或 <0.5% | 排查刷量或内容质量问题 |
| 线索断流 | 连续3天零线索 | 检查投放状态和落地页 |
| 数据缺失 | 某平台连续2天无数据 | 检查账号状态和API连接 |

## 输出格式
- **Markdown 分析报告**：核心结论 + 关键图表 + 建议
- **Excel 数据明细表**：原始数据 + 计算字段 + 条件格式
- **可视化图表**：折线图（趋势）、柱状图（对比）、热力图（时间分布）、漏斗图（转化路径）
