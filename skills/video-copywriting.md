# 视频文案创作技能 (Video Copywriting)

## 定位

汽车行业爆款短视频文案创作引擎。基于四品牌差异化定位，融合短视频平台算法逻辑，生成高转化、高传播的汽车短视频脚本。

---

## 1. 爆款短视频结构模型

### 1.1 黄金结构（3秒+主体+CTA）

```
┌──────────────────────────────────────────────┐
│  黄金3秒（Hook）                              │
│  → 制造悬念 / 数据冲击 / 视觉冲突 / 情感共鸣  │
├──────────────────────────────────────────────┤
│  内容主体（Body）                             │
│  → 痛点场景 → 解决方案 → 产品呈现 → 价值证明  │
├──────────────────────────────────────────────┤
│  行动号召（CTA）                              │
│  → 限时福利 / 试驾引导 / 互动引导 / 关注转化   │
└──────────────────────────────────────────────┘
```

### 1.2 各环节详细拆解

| 环节 | 时长 | 目标 | 技巧 |
|------|------|------|------|
| 黄金3秒 | 0-3s | 阻止划走 | 反问句、惊人数据、视觉冲击、冲突场景 |
| 痛点共鸣 | 3-8s | 引发共情 | 生活场景还原、使用痛点放大、对比冲击 |
| 产品展示 | 8-20s | 核心卖点 | 外观特写、内饰巡览、功能演示、参数对比 |
| 价值证明 | 20-30s | 建立信任 | 用户证言、销量数据、权威认证、实测视频 |
| 行动号召 | 30-35s | 驱动转化 | 限时优惠、试驾预约、评论区互动、关注引导 |

## 2. 四品牌文案风格差异化

### 2.1 广汽传祺 — 家庭实用派

**人群画像**：30-45岁家庭用户，注重空间、舒适性、性价比

**文案风格**：
- 语调：亲切温暖、实用主义
- 关键词：大空间、全家出行、省油耐用、智能安全
- 节奏：娓娓道来，场景化叙事
- 拍摄建议：家庭场景、接送孩子、周末出游

**示例开头**：
> "一家老小出门，最怕什么？空间不够、坐着不舒服、油耗又高..."
> "二胎家庭选车，这3个硬指标千万别踩坑..."

### 2.2 上汽奥迪 — 豪华品质派

**人群画像**：35-50岁高净值用户，注重品牌、品质、驾驶体验

**文案风格**：
- 语调：克制优雅、精英视角
- 关键词：品味、质感、驾驭、稀缺、传承
- 节奏：沉稳有力，留白和呼吸感
- 拍摄建议：光影质感、细节特写、都市夜景

**示例开头**：
> "有人说40万买奥迪不如买XX，我带你看3个细节..."
> "一辆真正的好车，不会告诉你它有多好，但每一个细节都在说..."

### 2.3 广汽昊铂 — 先锋科技派

**人群画像**：28-40岁科技爱好者和潮流引领者

**文案风格**：
- 语调：前卫创新、极客视角
- 关键词：智能、极速、颠覆、未来、黑科技
- 节奏：快速切换，视觉冲击，概念化表达
- 拍摄建议：赛博朋克风、实验室场景、光影特效

**示例开头**：
> "0-100加速1.9秒是什么概念？就是你还没反应过来就已经..."
> "为什么国产超跑的智能座舱，让百万豪车都沉默了..."

### 2.4 广汽埃安 — 年轻潮流派

**人群画像**：22-35岁年轻用户，注重个性、环保、智能互联

**文案风格**：
- 语调：活力热情、生活共鸣
- 关键词：潮玩、省钱、智能、颜值、环保
- 节奏：轻松明快，社交语言，梗和热词
- 拍摄建议：街头潮流、户外露营、网红打卡

**示例开头**：
> "月薪8000也能养得起的电动车，除了它我想不到第二个..."
> "这台车的配色，设计师怕不是个调色盘成精了..."

## 3. 文案模板库（10种模板）

### 模板1：数据冲击型

```
【开头】你知道吗？[惊人数据/事实]
【展开】没错，就是[产品名称]。[数据解读/场景还原]
【产品】它凭什么能做到？[卖点1] / [卖点2] / [卖点3]
【结尾】[价格/优惠]，你会考虑吗？评论区告诉我。
```

### 模板2：痛点共鸣型

```
【开头】[目标用户]，你是不是也遇到过[痛点场景]？
【共情】[痛点放大，列举具体困扰]
【方案】[产品名称]的[核心功能]，彻底解决这个问题。
【演示】你看[功能演示]...
【结尾】现在[行动引导]，别再[痛点结果]了。
```

### 模板3：对比揭秘型

```
【开头】[产品A]和[产品B]到底怎么选？3点说清楚。
【对比1】[维度A]：[A的成绩] vs [B的成绩]
【对比2】[维度B]：[A的成绩] vs [B的成绩]
【对比3】[维度C]：[A的成绩] vs [B的成绩]
【结论】如果你[需求]，闭眼入[推荐产品]。
```

### 模板4：场景体验型

```
【开头】[时间/地点]，开着[产品]去了[场景]...
【过程】[体验细节1] → [体验细节2] → [体验细节3]
【感受】说真的，[真实感受/意外发现]
【结尾】如果你也想[场景体验]，[行动引导]。
```

### 模板5：老板/销售视角型

```
【开头】做汽车[职业]这么多年，今天我决定说点真话...
【行业痛点】其实[行业内幕/潜规则]
【产品诚意】但[品牌/产品]的[做法]，让我刮目相看
【细节展开】[具体做法1/2/3]
【结尾】[行动引导]
```

### 模板6：车主证言型

```
【开头】[产品]开了[时长]，跑了[公里数]，说点真实感受。
【优点】[优点1/2/3]，确实没得黑。
【缺点】但[客观缺点]，该说还得说。
【总结】[整体评价]，适合[目标人群]。
```

### 模板7：热点蹭流型

```
【开头】最近[热点事件]火了，但其实汽车圈也有一个...
【关联】[事件/话题]和[产品卖点]的关联
【展开】[产品]的[相关卖点]到底有多[形容词]
【结尾】[互动引导]
```

### 模板8：教学干货型

```
【开头】[数字]个[品牌/车型]的隐藏功能，90%的车主不知道！
【功能1】[功能名]：[操作方法] → [效果展示]
【功能2】[功能名]：[操作方法] → [效果展示]
【功能3】[功能名]：[操作方法] → [效果展示]
【结尾】学会的点个赞，没学会的收藏慢慢看。
```

### 模板9：挑战测试型

```
【开头】今天我们来测一测[产品]的[极限指标]，到底有没有[官方数据]那么神？
【测试过程】[测试方法] → [测试过程] → [实时数据]
【结果】最后的结果是... [悬念揭晓]
【评价】[真实评价/对比分析]
```

### 模板10：情感故事型

```
【开头】[时间/场景]，[人物]做了一个决定...
【故事】[与产品相关的情感故事/生活片段]
【关联】也许[产品]不仅仅是一辆车，更是[情感价值]
【结尾】[共鸣引导/情感升华]
```

    def generate_script(self, template: str = "random", model: str = "",
                        duration: int = 35) -> dict:
        """
        生成短视频脚本

        参数:
            template: 模板类型 (data/pain/compare/scene/boss/owner/hot/tutorial/challenge/story/random)
            model: 车型名称
            duration: 视频时长(秒)，默认35秒

            "data", "pain", "compare", "scene", "boss",
            "owner", "hot", "tutorial", "challenge", "story"

        if template == "random" or template not in templates:
            template = random.choice(templates)

        brand_name = self.config["name"]
        model = model or f"{brand_name}主力车型"

        scripts = {
            "data": self._gen_data_impact(model),
            "pain": self._gen_pain_point(model),
            "compare": self._gen_compare(model),
            "scene": self._gen_scene(model),
            "boss": self._gen_boss_view(model),
            "owner": self._gen_owner_review(model),
            "hot": self._gen_hot_topic(model),
            "tutorial": self._gen_tutorial(model),
            "challenge": self._gen_challenge(model),
            "story": self._gen_story(model),
        }

        script = scripts.get(template, scripts["data"])

        return {
            "brand": brand_name,
            "model": model,
            "template": template,
            "duration_seconds": duration,
            "script": script,
            "generated_at": datetime.now().isoformat(),
        }
