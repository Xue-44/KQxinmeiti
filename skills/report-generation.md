# 报告生成技能

## 适用场景
- 新媒体运营日报/周报/月报
- 专项分析报告（投放复盘、活动总结、爆款拆解）
- 竞品深度分析报告
- 向管理层汇报的PPT演示文稿
- 季度/年度运营总结

## 标准流程
1. **需求确认**：报告类型、时间范围、目标受众、交付格式
2. **数据准备**：调用数据分析技能获取所需数据和图表
3. **结构组织**：按对应模板组织内容层次
4. **初稿撰写**：生成 Markdown 版本，数据驱动 + 观点清晰
5. **格式转换**：按需生成 PPTX / PDF / 飞书文档

## 报告类型与模板

### 1. 日报（每日 09:30 前）
```
结构：
- 昨日核心数据概览（4个品牌 × 3个核心指标）
- 亮点内容（播放量 Top 3，附链接）
- 异常告警（如有）
- 今日待办提醒
```
**目标受众**：新媒体团队内部
**篇幅**：一屏可读完，飞书卡片消息

### 2. 周报（每周一 10:00）
```
结构：
- 本周整体数据总览（环比变化）
- 各品牌表现拆解
  - 核心指标达成情况
  - 爆款内容分析（Top 5）
  - 低效内容诊断（Bottom 3）
- 竞品动态摘要
- 下周内容计划概览
- 待决策事项
```
**目标受众**：部门负责人 + 管理层
**篇幅**：Markdown 约 2000 字 + 图表

### 3. 月报（每月1日 10:00）
```
结构：
- 月度核心指标达成率（vs KPI）
- 各品牌深度分析
  - 粉丝增长曲线
  - 内容效果分布（播放量区间分布）
  - 投放ROI分析
  - 团队产能分析（人均产出）
- 竞品月度评分排名
- 行业趋势观察
- 下月策略建议
- 预算使用情况
```
**目标受众**：管理层 + 集团汇报
**篇幅**：Markdown 约 5000 字 + 丰富图表

### 4. 专项分析报告
```
结构：
- 背景与目标
- 数据概览
- 深度分析（多维度交叉）
- 核心发现（3-5条）
- 优化建议（可落地、有时限）
- 附录（原始数据 + 方法说明）
```

### 5. 管理层汇报PPT
```
结构（10-15页）：
1. 封面（报告主题 + 时间范围）
2. 核心结论（一页总览）
3-8. 各维度数据详析（每页一个核心观点 + 图表支撑）
9. 竞品对比
10. 问题与挑战
11. 下阶段计划
12. 需要支持的资源
```

## 数据可视化规范

### 图表选择原则
| 数据类型 | 推荐图表 | 示例场景 |
|---------|---------|---------|
| 时间趋势 | 折线图 | 粉丝增长、播放量变化 |
| 横向对比 | 柱状图/条形图 | 各品牌互动率对比 |
| 占比分布 | 饼图/环形图 | 内容形式分布 |
| 时间分布 | 热力图 | 发布时段 × 星期效果 |
| 转化路径 | 漏斗图 | 曝光→点击→线索→到店→成交 |
| 多维度对比 | 雷达图 | 竞品综合评分 |

### 配色方案
- **主色**：品牌色（传祺蓝 #005BAC、奥迪黑 #000000、昊铂紫 #6B3FA0、埃安绿 #00B894）
- **辅色**：中性灰 #6B7280、警示红 #EF4444、成功绿 #10B981
- **背景**：白色/浅灰，避免深色背景（打印友好）

### 图表标注规范
- 所有图表必须标注数据来源和时间范围
- 环比/同比变化用红色（下降）和绿色（上升）标注
- 异常数据点用红色虚线框标记

## PPT 生成规范

### 页面布局
- 16:9 宽屏比例
- 标题区（顶部 15%）+ 内容区（中部 75%）+ 页码区（底部 10%）
- 每页不超过 3 个核心要点
- 图表占页面面积 60% 以上

### 文字规范
- 标题：28pt 加粗
- 正文：18pt 常规
- 数据标注：14pt
- 每页文字不超过 80 字

### 品牌模板
- 传祺汇报：传祺品牌模板（蓝白配色）
- 奥迪汇报：奥迪品牌模板（黑白配色）
- 集团汇报：铠祺集团通用模板（深蓝配色）

## 质量检查清单
- [ ] 所有数据已核验，无计算错误
- [ ] 环比/同比方向正确（上升=绿，下降=红）
- [ ] 图表标注完整（标题、坐标轴、数据来源、时间范围）
- [ ] 结论有数据支撑，建议可落地
- [ ] 无错别字和格式问题
- [ ] PPTX/PDF 导出后逐页检查排版

## 输出格式
| 场景 | 主格式 | 辅格式 | 交付渠道 |
|------|--------|--------|---------|
| 日报 | 飞书卡片消息 | — | 飞书群 |

| 月报 | PPTX | PDF | 飞书文档 + 群文件 |
| 专项报告 | Markdown | PPTX（如需汇报） | 飞书文档 |
| 管理层汇报 | PPTX | PDF | 飞书文档 + 会议投屏 |
