# 昊铂埃安BU 5-12月昊铂融合店垂媒支持政策报告

**发布单位**：昊铂埃安 BU 用户发展部  
**发布日期**：2026年5月3日  
**文件类型**：内部工作指引（请勿外传）

---

## 一、政策背景

为推进双品牌渠道全面融合，促进新车终端销量实现突破性提升，强化经销商门店公域流量获取能力、线索运营质量及终端转化效率，实现品牌声量与终端销量双提升。

---

## 二、支持周期与范围

| 项目 | 说明 |
|------|------|
| **支持周期** | 2026年5月至12月 |
| **核销频次** | 全年度共3次（5-6月、7-9月、10-12月） |
| **支持门店** | 昊铂品牌已取得渠道融合双品牌认证（埃安品牌）的门店 |
| **支持平台** | 懂车帝、汽车之家、易车 |

---

## 三、核心支持政策

### 3.1 支持比例

| 平台 | 单品牌 | 双品牌增开部分 |
|------|--------|----------------|
| 懂车帝 | 原有品牌支持政策 | 统一支持 **52%** |
| 汽车之家 | 原有品牌支持政策 | 统一支持 **52%** |
| 易车 | 原有品牌支持政策 | 统一支持 **52%** |

### 3.2 核销前提条件

**月度/季度提车要求：**
- 月度提车总数 ≥ **2台**
- 季度提车总数 ≥ **6台**

**例外规则：**
- 5-6月：提车总数≥4台可核销；若<4台但6月≥2台，可进行6月月度核销
- 7-9月：季度≥6台可核销；若<6台但8月≥2台，可进行8月月度核销

**在线要求：**
- 季度末最后一天前下线的平台，当季度不予核销
- 开通不足一个月：按天数折算（5月全额/当月总天数×在线天数；6月及以后：全额/当月总天数×在线天数×80%）
- 销售店退网后即要求下线，厂家不继续补贴

### 3.3 支持方式

经销商开通平台并提交核销资料（付款通知书、合同、发票等），审核通过后补贴费用支付至**经销商折扣池**。

## 五、双品牌开通说明

- **汽车之家/易车**：昊铂品牌开通后，可**免费添加3款埃安品牌车系**（5月起调整为**i60、N60、RT车系**）
- 若想开通全部车系，需开通双品牌，增开部分按双品牌费用及支持政策执行
- **懂车帝**：按门店可售卖埃安车系进行添加，联系区域新媒体经理报备开通
